## Requirement and depedencies
- Ruby version: ruby 2.4.1

## Get started
- check out source code
- run `bundle`

## Run test
- run `rails t`


## Start services
- run `rake db:migrate`
- run `rails s`

Services will be running at `http://localhost:3000`
